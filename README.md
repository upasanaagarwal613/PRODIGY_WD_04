# Personal Portfolio Website

This project is a responsive personal portfolio website built using HTML, CSS, and JavaScript. The website showcases my skills, education, and projects as a web developer.

## Features

- Responsive design for mobile and desktop
- Modern UI with animations
- Education section with circular progress indicators
- Skills section including technical and personal skills
- Projects section to showcase my work
- Contact section for communication

## Technologies Used

- HTML
- CSS
- JavaScript

## Projects Included

- Tic Tac Toe Game
- Stopwatch Web Application
- Ticket Booking Website
- Personal Portfolio Website
